# Pygame-Project

Requirements for running the game:
Python 3.8.3, pygame 2.0.0, the following libraries random, os and Vector2 from library pygame.math.

Also, an additional requirement is a Windows 10 or a latest update so that all the features of the game can work properly. 

To play the game, run the MAIN.py, from the folder "WhiteSnake" and use the up,down,left and right keys to move the spaceship
and the space key to shoot bullets.
